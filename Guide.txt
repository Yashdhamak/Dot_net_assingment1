# Yash Dhamak 

TaskTracker Project
Description
TaskTracker is a console-based task management application developed in C#. It provides users with the ability to manage tasks efficiently by allowing them to add, read, update, and delete tasks through a simple menu-driven interface.

Features

Add Task: Users can add tasks by providing a title and description.
Read Tasks: Users can view all existing tasks.
Update Task: Users can modify the title and description of existing tasks.
Delete Task: Users can remove tasks from the list.
User-Friendly Interface: The application offers an intuitive menu-driven interface for easy navigation and interaction.
Technologies Used

C#
.NET Framework


Additional Notes

Ensure you have .NET Framework installed on your system.
Refer to the project documentation or troubleshoot section for any issues during installation or usage.
TaskTracker simplifies task management with its straightforward interface and essential features, making it an ideal choice for personal or small-scale task tracking needs.
